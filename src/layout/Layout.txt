공통 UI 구조 관리
    모든 페이지에서 반복되는 UI를 한 곳에서 관리한다.

    Header
    Main Content
    Footer

    이 중에서
        👉 Header를 공통으로 출력한다.

children의 의미
    children은 Layout 안에 들어오는 페이지 컴포넌트를 의미한다.

실제 렌더링 구조
Layout
│
├── Header
│
└── main
      └── children (페이지 내용)


Fragment (<> </>) 역할
    React Fragment는
        👉 불필요한 div를 만들지 않기 위해 사용

marginTop: “60px” 이유
    Header가 보통
        position: fixed;
    로 고정되어 있기 때문에
        👉 Header 높이만큼 공간을 확보해야 한다.
    그래서
        Header height = 60px
    만큼 내려준다.

    3️⃣ 전체 웹 페이지 기본 틀 제공
    4️⃣ 코드 중복 제거
