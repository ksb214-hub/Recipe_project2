SearchPage 설계 문서

1. 개요 (Overview)
  SearchPage는 사용자가 레시피를 검색할 수 있는 페이지이다.
  사용자는 검색창에 재료명 또는 요리 이름을 입력하여 레시피를 찾을 수 있다.
  검색 결과는 Card 컴포넌트를 활용하여 리스트 형태로 출력된다.

2. 개발 환경
    항목		내용
	Framework	React
	Language	JavaScript
	Routing		React Router
	Component	Input, Button
	Styling	    CSS


3. 폴더 구조

		SearchPage.js → 검색 기능 및 UI 구조
		SearchPage.css → 검색 페이지 스타일

4. SearchPage 목적
  SearchPage는 다음 기능을 제공한다.
    레시피 검색
    검색 결과 출력
    레시피 카드 표시

11. Card 출력 구조
  검색 결과는 Card 컴포넌트를 사용하여 출력된다.
  
  예시
    {results.map((recipe) => (
      <Card
        key={recipe.id}
        title={recipe.title}
        description={recipe.description}
        time={recipe.time}
        difficulty={recipe.difficulty}
        servings={recipe.servings}
      />
    ))}
