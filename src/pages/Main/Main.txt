Main 페이지 설계 문서

1. 개요 (Overview)
      Main 페이지는 zero-naeng-fe 서비스의 메인 화면을 제공하는 페이지이다.

      사용자는 메인 페이지에서 추천 레시피(BEST 요리모음) 를 확인할 수 있으며,
      각 레시피는 Card 형태로 표시된다.

      또한 상단에는 Header 컴포넌트를 통해 네비게이션 기능이 제공된다.

2. 개발 환경
      항목		        내용
      Framework	      React
	Language	     JavaScript
	Routing		React Router
	Component	      Input, Button
	Styling	        CSS

      구성 요소

4. 목적

      Main 페이지는 다음 기능을 제공한다.
      - 추천 레시피 목록 제공
      - 레시피 카드 UI 제공
      - 레시피 상세 페이지 이동 (추후 기능)

7. 데이터 흐름
      Main 페이지의 핵심 구조는 데이터 기반 UI 렌더링이다.
            plaintext
                  recipes 데이터
                        │
                        ▼

                        │
                        ▼
                  Card 컴포넌트 전달
                        │
                        ▼
                        UI 출력

8. Card 출력 구조
      레시피 목록은 Card 컴포넌트를 사용하여 출력한다.

      예시
            JavaScript
                  recipes.map((recipe) => (
                        <Card
                        key={recipe.id}
                        title={recipe.title}
                        image={recipe.image}
                        />
                  ))
