Login.js + Login.css
폴더 구조 
src/
│
├── App.js (Router)
│
├── Main/
│   └── Main.js
│
├── Login/
│   ├── Login.js
│   └── find_id/
│       └── find_id.js
│
├── SearchPage/
│   ├── SearchPage.js
│   └── SearchPage.module.css

목적: 레시피 로그인화면을 구현하는 것을 목표로 한다.
사용언어: react를 사용하고 언어는 js를 사용한다.

기능
│
├── 입력
│     ├── 아이디 입력 (id state)
│     └── 비밀번호 입력 (pw state)
│
├── 로그인 처리
│     └── handleSubmit()
│           ↓
│        로그인 요청 (API 예정)
│
├── 이동 기능
│     ├── 아이디 찾기 → /find-id
│     ├── 비밀번호 찾기 → /find-pw
│     └── 회원가입 → /join (추후)


구조 흐름
Login
│
├── Input (아이디)
├── Input (비밀번호)
├── Button (로그인)
└── Link (아이디 찾기 / 비번 찾기 / 회원가입)

데이터 흐름
	input 입력
		↓
	useState 저장
		↓
	handleSubmit 실행
		↓
	로그인 처리

Login 페이지는
“Input + Button + Router 연결” 구조
